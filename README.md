# JavaPrograms
1️⃣ The Hello World program demonstrates the basic structure of a Java application and shows how execution begins from the main method to display output on the console.
2️⃣ The data types program illustrates the declaration and initialization of various primitive data types and displays their values to explain how data is stored and used in Java.
3️⃣ The multiplication table program demonstrates the use of a for loop to generate and display the multiplication table of a given number up to 12.
